## Автотесты для API
В данной работе покрыты тестами API на создание пользователя, авторизацию пользователя и создание заказа. Так же сделать отчет в Allure.
## Используемые технологии
- Java 11
- JUnit 4.13.2
- RestAssured 5.5.6
- Gson 2.13.2
- Allure
## Способ запуска
- Чтобы получить интерактивный веб-отчет Allure нужно:
1. Открыть окно инструмента Maven
2. Во вкладке Lifecycle последовательно двойным щелчком запустить clean и test
3. Во вкладке Plugins-allure двойным щелчком запустить allure:report и allure:serve
- Или выполнить следующие команды через терминал:
1. mvn clean test allure:report - чтобы запустить тесты и сформировать отчет
2. mvn allure:serve - запустить интерактивный сервер Allure
