## Автотесты для UI
В данной работе написаны тесты для тестирования UI: проверка успешной регистрации, появления сообщения при неверном пароле при регистрации, кнопки входа в разных частях и переключение между разделами конструктора. Так же сделать отчет в Allure.
## Используемые технологии
- Java 11;
- Maven 4.0.0;
- JUnit 4.13.2;
- Selenium 4.40.0;
- RestAssured 5.5.6;
- Gson 2.13.2;
- Allure.
# Перед запуском!!!
1. Скачать Яндекс Драйвер "https://github.com/yandex/YandexDriver" (Версия драйвера должна соответствовать версии браузера);
2. Перейти в класс YandexPath в пакете util;
3. Указать свой путь до Яндекс Браузера и Яндекс Драйвера.
## Способ запуска
- Чтобы запустить тесты через Chrome, в терминале нужно ввести команду "mvn clean test";
- Чтобы запустить тесты через Яндекс Браузере, в терминале нужно ввести команду "mvn clean test -Dbrowser=yandex".

Чтобы получить интерактивный веб-отчет Allure нужно:
1. Открыть окно инструмента Maven;
2. Во вкладке Plugins-allure двойным щелчком запустить allure:report и allure:serve.

Или выполнить следующие команды через терминал:
1. mvn allure:report - чтобы сформировать отчет;
2. mvn allure:serve - запустить интерактивный сервер Allure.
