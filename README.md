# -Ключевые проекты:
## Юнит‑тестирование сервиса Stellar Burgers (Unit-tasts)
Проделанная работа: написал набор юнит‑тестов для бизнес‑логики сервиса, использовал мокирование зависимостей для изоляции тестируемых компонентов.
Результаты: достиг 100 % покрытия кода (по метрике JaCoCo) для ключевых модулей, что снизило вероятность критических сбоев.
Инструменты: Java, Maven, JUnit4, Mockito, JaCoCo.

## Автоматизация API‑тестирования для Stellar Burgers (API-tests)
Проделанная работа: создал фреймворк для автоматизированного тестирования REST API, реализовал сценарии проверки эндпоинтов, валидации JSON‑ответов и обработки кодов HTTP‑статусов.
Результаты: сократил время ручного регрессионного прогона API‑тестов, настроил генерацию отчётов Allure для наглядного отображения результатов тестов.
Инструменты: Java, Maven, JUnit4, RestAssured, Gson, Allure.

## Автоматизация UI‑тестирования для Stellar Burgers (UI-tests)
Проделанная работа: спроектировал и внедрил набор UI‑автотестов на Selenium, покрыл ключевые пользовательские сценарии.
Результаты: обеспечил стабильное регрессионное тестирование интерфейса, снизил количество регрессных багов в продакшене на 25 %.
Инструменты: Java, Maven, JUnit4, Selenium, RestAssured, Gson, Allure.
